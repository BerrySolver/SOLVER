# 💰 포인트
> 다양한 활동을 통해서 포인트를 얻을 수 있으며, 그 모은 포인트를 통해서 프로필에 노출하고 메인 화면에서 표현하는 등의 어필을 할 수 있습니다.

|          사유           | 포인트 |
| :---------------------- | :----: |
| 첫 답변                 |  +50p  |
| 답변 채택               | +100p  |
| 화상 답변               | +100p  |
| 만료 12시간 이내의 답변 |  +50p  |
| 신고                    | -100p  |
